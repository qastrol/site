📢 Hoe werkt de stream? 📢

Tussen elke game is een poll te zien waarin kan worden gestemd op de volgende visgame.
Met !request [gamenaam] kan je een game toevoegen aan de volgende poll! Er is een limiet op twee suggesties, maximaal 1 per gebruiker.
Wil je je suggestie weghalen? Doe dan !request delete.
Met !voteskip kan je een stemming voor de volgende game forceren.
